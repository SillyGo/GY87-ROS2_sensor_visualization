# GY87-ROS2_sensor_visualization
in this short but quite fun to develop repository, i take a GY87 IMU sensor, make an arduino code to read data from it and send it via Serial monitor, then have a python script take that serial data, convert it into a ROS2 topic publishing IMU messages, and then visualize the data in RVIZ2. This project is to used in odometry in the ROBOCAR project
